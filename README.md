# 谷文昌纪念馆 · 红色文化数字化网页展厅

一个沉浸式的红色文化数字化网页展厅，展示谷文昌同志（1915-1981）的生平事迹与精神传承。

## 展厅内容

- **序厅** — 主题标语与粒子动画
- **生平历程** — 1915—2015 交互式时间线
- **丰功伟绩** — 治沙造林、兵灾家属、兴修水利、廉洁奉公
- **东山巨变** — 旧貌 vs 新颜对比
- **精神丰碑** — "四有"精神（心中有党/民/责/戒）
- **数字文物** — 纪念馆实景照片与历史档案
- **谆谆教诲** — 经典语录墙
- **薪火相传** — 数据统计与致敬

## 技术特点

- 纯前端单文件（HTML/CSS/JS），无需后端
- 响应式设计，适配手机/平板/桌面端
- 滚动揭示动画、粒子效果、交互式时间线
- 历史照片来自新华社及谷文昌纪念馆公开资料

## 使用方法

直接用浏览器打开 `index.html` 即可，或访问 GitHub Pages 在线预览。
